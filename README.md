# multi_parser

About
=====
Parser for different services with threads and queues.


Work with youtube, songlyrics.com


Installation
============
```
git clone https://github.com/Ferlony/multi_parser
cd multi_parser
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```


Usage
=====
```
cd multi_parser
source venv/bin/activate
python main.py
```
