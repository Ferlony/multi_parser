# multi_parser

About
=====
Parser for different services with threads and queues.


Works with youtube, songlyrics.com, selenium
Has config file


Installation
============
```
git clone https://github.com/Ferlony/multi_parser
cd multi_parser
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```


Usage
=====
```
source venv/bin/activate
python -m src.main
```
